Project Overview

Hiring Heatmap is a data analytics platform designed to visualize real-time hiring trends, skills demand, job market competition, and recruitment patterns across industries and locations. The objective of this internship project was to design and implement a complete data analytics workflow—from mock dataset generation to segmentation modeling, predictive analytics, dashboards, and investor-ready insights.

Key Deliverables

This handover includes all artifacts created throughout the project, organized to support clarity and reproducibility.

Category	Description
Documentation	Market analysis, A/B testing, scoring frameworks, schema plans, investor materials
Notebooks	Data simulation, skill analysis, segmentation, LTV prediction, investor EDA
Datasets	Mock datasets generated from analytics schema
Power BI Dashboards	Job demand analytics, segmentation strategy, product usage prototype
Presentations	Market intelligence summary, product roadmap, investor appendix

How to Reproduce

Open Google Colab notebooks in numerical order and run Runtime > Restart & Run All

Ensure datasets in 03_Datasets are uploaded or linked correctly

For Power BI dashboards, refresh BigQuery connections

Review slides in 05_Presentations for investor storytelling insights

Refer to the final handover summary Google Doc for master navigation links
